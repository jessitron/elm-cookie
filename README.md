# Elm Cookie

This is a wee Elm library that gives you access to cookies, through
Effects.

how to use cookies: http://www.quirksmode.org/js/cookies.html
how to use Effects: [the Elm Architecture](https://github.com/evancz/elm-architecture-tutorial/#example-5-random-gif-viewer)

Example project that uses this library:
[Try Elm Cookie](http://github.com/jessitron/try-elm-cookie)
